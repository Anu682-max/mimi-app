## Screenshot Requirements

### Required Screenshots
1. **Welcome/Onboarding** - Multi-language
2. **Profile View** - Showing verified badge
3. **Discover/Swipe** - AI match percentage
4. **Chat** - With translation toggle visible
5. **Match Animation** - "It's a Match!" in locale
6. **Settings** - Language selector

### Localization Notes
- All UI text must be in target language
- Include local phone frames (if required)
- Time format: 12hr (US) vs 24hr (JP/EU)
- Date format: MM/DD (US) vs DD/MM (EU) vs YYYY/MM/DD (JP)

## Checklist

### Pre-Launch
- [ ] Translate app name
- [ ] Translate subtitle/short description
- [ ] Translate full description
- [ ] Update keywords for locale
- [ ] Create localized screenshots
- [ ] Create localized preview video
- [ ] Set up localized support URL
- [ ] Set up localized privacy policy URL

### Post-Launch
- [ ] Monitor reviews in each language
- [ ] Translate "What's New" for updates
- [ ] Update ASO keywords based on local search trends
- [ ] A/B test different descriptions
